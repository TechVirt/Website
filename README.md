COOL
===========



[COOL](http://cool-framework.com/), A complete-packaged, modern responsive front-end framework based on Material Design for developing awesome projects on the web.

### Current Version : v1.0.0

## Sass Requirements:
- Ruby Sass 3.3+, LibSass 0.6+

## Supported Browsers:
Chrome 35+, Firefox 31+, Safari 7+, IE 10+
